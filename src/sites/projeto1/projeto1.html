Conteudos:



HTML5 é a quinta e mais recente versão da linguagem de marcação de hipertexto (HTML), usada para estruturar o conteúdo de páginas web.

CSS, ou Folhas de Estilo em Cascata, é uma linguagem usada para descrever a apresentação de documentos HTML ou XML. Em termos simples, o CSS define como os elementos de uma página web devem ser exibidos, incluindo cores, fontes, layout e outros aspectos visuais.

O que é a propriedade Display em css


A propriedade display determina como um elemento HTML é renderizado na página. Ela define se o elemento se comporta como um bloco, um elemento em linha ou se é ocultado. A propriedade display é fundamental para o layout e a disposição dos elementos em uma página web.

Display Caracteristicas
Block Faz com que um elemento HTML seja renderizado como um bloco, ocupando toda a largura disponível do elemento pai e iniciando em uma nova linha.
inline É exibido na mesma linha que outros elementos, ocupando apenas o espaço necessário para seu conteúdo.
Inline-Block Faz com que um elemento seja tratado como um bloco, mas ainda mantenha o comportamento de um elemento inline, permitindo que vários elementos sejam exibidos lado a lado e que propriedades como largura e altura sejam aplicadas.
Flex é usado para criar um layout flexível, permitindo que os elementos filhos se ajustem e se alinhem de forma dinâmica dentro do contêiner pai.
